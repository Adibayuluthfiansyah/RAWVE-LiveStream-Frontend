# Support

Need help with the RAWVE Frontend? We're here to help!
Please read the following guidelines to get the best support possible.

---

## 📖 Documentation

Before asking a question or submitting an issue, check the [README.md](README.md) for setup instructions, tech stack details, and architecture overview.

---

## 🐛 Bug Reports

Found a bug — UI glitches, video player errors, responsive design issues, or React hydration errors? Please open an issue on [GitHub Issues](https://github.com/adibayuluthfiansyah/rawve-frontend/issues).

When submitting a bug report, please include:

1. **Environment** — your OS and browser version
2. **Reproduction steps** — exact steps to reproduce the bug
3. **Expected vs actual behavior** — what you expected vs what happened
4. **Screenshots or recordings** — highly recommended for UI issues
5. **Console logs** — any error logs from the browser console

---

## 💡 Feature Requests

Have an idea to improve the viewing experience, Live Chat, or Creator Dashboard? Open a GitHub issue, use the `enhancement` label, and describe your idea and use case in detail. We'd love to hear it!

---

## 🔒 Security Issues

If you find a security vulnerability (such as XSS in the chat), **do not open a public issue.**
Refer to the [Security Policy](SECURITY.md) for instructions on how to report it privately.
